## Overview
This file is the raw Dexcom Clarity event log across study subjects. It contains high-frequency CGM records such as glucose measurements, calibration events, and alerts.

## Interpretation of event types

### 1. EGV
This is the dominant event type.

Likely meaning:
- Estimated Glucose Value
- routine CGM glucose reading generated by Dexcom

Implication:
- this is the core time series for glucose trend analysis
- it supports reconstruction of glycemic trajectories over time

### 2. Calibration
Likely meaning:
- calibration-related event

Implication:
- indicates periods where sensor calibration activity occurred
- can be relevant when assessing CGM performance and data quality

### 3. Alert
Likely meaning:
- alarm or threshold-related event

Examples may include:
- High
- Low
- Rise
- Fall
- Signal Loss
- Urgent Low
- Urgent Low Soon

Implication:
- useful for identifying clinically relevant glycemic events
- can support alarm burden analysis

### 4. Device
Likely meaning:
- device metadata or setup event

Implication:
- limited in count, but may help identify device context
